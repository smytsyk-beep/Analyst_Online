# OmniDash Mini Brand Guide

## 1. Brand essence

**OmniDash** is a modern analytics brand for e-commerce and performance marketing.  
Visual direction: clean SaaS, trustworthy data, growth, clarity, premium simplicity.

Core feeling:

- analytical
- modern
- optimistic
- high-signal
- product-led

## 2. Logo system

### Primary logo

Use the full **OmniDash** wordmark with the circular icon in:

- website header
- hero section
- presentations
- social cards
- proposals

### Icon / pictogram

Use the circular icon alone for:

- favicon
- app icon
- browser tab
- compact UI placements
- social avatar

### Safe area

Keep free space around the logo equal to at least **0.5x** of the icon diameter.

### Minimum size

- Full logo: **180 px** width minimum for digital use
- Icon only: **24 px** minimum
- Favicon: use dedicated PNG/ICO exports

### Neutral UI

- **Soft Background** — `#F3F6FA`

## 4. Typography

### Recommended brand font

**Inter**

Suggested usage:

- Headings: `Inter 700 / 800`
- Body text: `Inter 400 / 500`
- KPI cards: `Inter 600 / 700`

### Fallback stack

```css
font-family: 'Inter', 'Manrope', 'Segoe UI', Arial, sans-serif;
```

## 5. Visual style rules

Use:

- white or very light backgrounds
- blue-driven charts
- thin grid lines
- soft shadows
- rounded cards
- lots of breathing room
- minimal decorative noise

Avoid:

- heavy gradients behind the logo
- dark cluttered backgrounds
- too many competing accent colors
- condensed fonts
- glossy or skeuomorphic effects

## 6. Chart styling direction

Recommended chart colors:

- Primary line: `#4EC6D8`
- Primary bars: `#2F6EDB`
- Secondary bars: `#3F86E8`
- Positive accent: `#63C05A`
- Highlight accent: `#A6D63A`

Recommended chart treatment:

- line + bar combination
- subtle grid
- large numeric KPIs
- labels with high contrast
- low visual overload

## 7. Usage examples

### Good use

- white header with full logo
- favicon with icon only
- OG image with logo + chart composition
- dashboard cards using blue/cyan accents

### Incorrect use

- stretching logo horizontally
- changing wordmark colors randomly
- putting icon on busy photo backgrounds
- rotating the icon
- using different font for "Omni" and "Dash" without a brand update
